Вот краткое техническое задание (ТЗ) для создания учебной веб-страницы, демонстрирующей интерференцию от двух когерентных точечных источников света.

Техническое задание
Наименование проекта: Визуализация интерференционной картины от двух когерентных источников света
Среда реализации: Replit (HTML, CSS, JavaScript)
Тип проекта: Учебная интерактивная веб-страница для визуализации физических явлений

1. Цель работы
Создать интерактивную веб-страницу, демонстрирующую интерференционную картину двух когерентных точечных источников света с возможностью изменения параметров для наглядного изучения влияния частоты и геометрии на вид интерференционной картины.

3. Требования к функциональности
Отображение двумерной интерференционной картины с плавным градиентом интенсивности (например, с использованием Canvas API).

Интерфейс должен содержать:

Ползунок или ввод числа для частоты / длины волны

Ползунок для расстояния между источниками

Ползунок для расстояния до экрана

Кнопку «Сброс» для возвращения параметров к значениям по умолчанию

Отображение текущих значений параметров.

Интерактивное обновление графики при изменении параметров.

Возможность масштабирования области визуализации (zoom и панорамирование, опционально).

4. Требования к реализации
Используемый стек: HTML + CSS + JavaScript (Canvas или WebGL при необходимости).

Все расчёты и визуализация выполняются на клиенте.

Структура проекта:

index.html — структура страницы

style.css — оформление интерфейса

script.js — визуализация и обработка входных параметров

Код должен быть документирован и пригоден для учебных целей.

Совместимость: браузеры Chrome, Firefox, Edge.

Среда: Replit.

5. Дополнительные пожелания
Цветовая схема: чёрный фон с интерференционными полосами в бело-синих или RGB-оттенках.

Возможность отображения линий равной фазы (опционально).

Всплывающие подсказки при наведении на элементы интерфейса с объяснением физического смысла параметров.
