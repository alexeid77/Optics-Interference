# WaveOptics — Визуализация интерференционной картины

## Описание проекта

**WaveOptics** — это интерактивное веб-приложение для визуализации интерференционной картины, возникающей от двух когерентных точечных источников света (эксперимент Юнга с двумя щелями). Приложение разработано в образовательных целях для наглядного изучения волновых свойств света и влияния различных параметров на интерференционную картину.

## Основные возможности

### Интерактивная визуализация
- Двумерная интерференционная картина отображается в реальном времени на HTML5 Canvas
- Плавный градиент интенсивности с цветовой схемой, соответствующей выбранной длине волны
- Мгновенное обновление при изменении любого параметра

### Настраиваемые параметры

| Параметр | Диапазон | Описание |
|----------|----------|----------|
| **Длина волны (λ)** | 380 — 750 нм | Определяет цвет света и ширину интерференционных полос |
| **Расстояние между источниками (d)** | 0.05 — 5.0 мм | Влияет на количество и ширину полос |
| **Расстояние до экрана (L)** | 10 — 200 см | Изменяет масштаб интерференционной картины |

### Сохранение конфигураций
- Возможность сохранять текущие настройки эксперимента с произвольным названием
- Загрузка ранее сохранённых конфигураций из базы данных
- Удаление ненужных сохранённых настроек

### Пользовательский интерфейс
- Современный тёмный интерфейс в научном стиле
- Всплывающие подсказки с объяснением физического смысла каждого параметра
- Адаптивный дизайн для различных размеров экрана
- Кнопка сброса параметров к значениям по умолчанию

## Физическая модель

Расчёт интенсивности света в каждой точке экрана основан на классической формуле интерференции:

$$I(x, y) = I_0 \cdot \left(1 + \cos\left(\frac{2\pi \Delta r}{\lambda}\right)\right)$$

где:
- $I_0$ — начальная интенсивность
- $\Delta r = |r_2 - r_1|$ — разность хода волн от двух источников до точки наблюдения
- $\lambda$ — длина волны света
- $r_1, r_2$ — расстояния от каждого источника до точки на экране

### Геометрия эксперимента
- Два когерентных точечных источника расположены симметрично относительно оси на расстоянии $d$ друг от друга
- Экран наблюдения находится на расстоянии $L$ от источников
- Интерференционная картина рассчитывается для каждого пикселя экрана

## Технологический стек

### Frontend
- **React** — библиотека для построения пользовательского интерфейса
- **TypeScript** — типизированный JavaScript
- **Canvas API** — высокопроизводительная отрисовка графики
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **shadcn/ui** — компоненты пользовательского интерфейса
- **TanStack Query** — управление состоянием и кэширование запросов

### Backend
- **Express.js** — веб-фреймворк для Node.js
- **PostgreSQL** — реляционная база данных
- **Drizzle ORM** — современный ORM для TypeScript

## Запуск приложения

Приложение запускается автоматически командой:

```bash
npm run dev
```

После запуска приложение доступно по адресу, указанному в консоли.

## Образовательная ценность

Данное приложение позволяет:

1. **Наглядно изучить** явление интерференции света
2. **Исследовать зависимость** ширины интерференционных полос от:
   - Длины волны света
   - Расстояния между источниками
   - Расстояния до экрана наблюдения
3. **Понять связь** между цветом света и длиной волны в видимом диапазоне
4. **Экспериментировать** с различными комбинациями параметров
5. **Сохранять и сравнивать** результаты различных экспериментальных настроек
